# 🚀 IntelliChat AI

**IntelliChat AI** is a modern full-stack AI assistant designed to deliver intelligent conversations, image understanding, and AI-powered image generation — all within a sleek and responsive interface.

Built using **Node.js, Express.js, and Vanilla JavaScript**, IntelliChat combines powerful AI integrations with a polished user experience to create a seamless multi-modal assistant capable of chatting, analyzing images, and generating visuals from text prompts.

Whether you're exploring conversational AI, visual intelligence, or creative image generation, **IntelliChat AI** provides an interactive and real-time AI experience.

---

# ✨ Key Features

💬 **Context-Aware AI Chat**
Engage in intelligent multi-turn conversations with persistent session-based context powered by OpenRouter AI models.

🖼️ **AI Image Analysis**
Upload images and receive detailed AI-generated descriptions and visual analysis using vision-capable AI models.

🎨 **Text-to-Image Generation**
Generate unique AI images from text prompts using integrated image generation APIs.

⚡ **Real-Time Interactions**
Experience smooth and responsive communication with asynchronous API handling and dynamic UI updates.

🌙 **Modern UI/UX**
Clean dark-themed interface with responsive layouts, smooth interactions, and intuitive navigation.

🔗 **Multi-API Integration**
Seamlessly integrates multiple AI services for conversational intelligence, vision analysis, and image generation.

# 🧠 Tech Stack

| Layer             | Technology                      |
| ----------------- | ------------------------------- |
| **Frontend**      | HTML5, CSS3, Vanilla JavaScript |
| **Backend**       | Node.js, Express.js             |
| **Architecture**  | MVC Architecture, REST APIs     |

# ⚙️ Installation

## Clone the Repository

```bash id="s24mha"
git clone https://github.com/akshitaa011/IntelliChat-AI.git
cd IntelliChat-AI
```

## Install Dependencies

```bash id="f1ixfa"
npm install
```

## Create a `.env` File

```env id="u4tjlwm"
OPENROUTER_API_KEY=your_openrouter_api_key
PORT=3000
```

## Run the Application

```bash id="sx7hq8"
node app.js
```

## Open in Browser

```txt id="vjlwmw"
http://localhost:3000
```
