# IntelliChat AI

A full-stack AI assistant built with Node.js, Express.js, and Vanilla JS.

## Features
- **Chat** — Context-aware conversations via OpenRouter API
- **Image Analysis** — Vision AI to analyze uploaded images
- **Text-to-Image** — Generate images from text prompts via Pollinations AI

## Tech Stack
- **Backend:** Node.js, Express.js, REST APIs, MVC architecture
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **APIs:** OpenRouter (LLM), Pollinations AI (image generation)

## Setup

1. Clone the repo
   \`\`\`bash
   git clone https://github.com/akshitaa011/IntelliChat-AI.git
   cd IntelliChat-AI
   \`\`\`

2. Install dependencies
   \`\`\`bash
   npm install
   \`\`\`

3. Create a \`.env\` file
   \`\`\`env
   OPENROUTER_API_KEY=your_openrouter_key_here
   PORT=3000
   \`\`\`

4. Run the app
   \`\`\`bash
   node app.js
   \`\`\`

5. Open http://localhost:3000
